## Development

**Clone this repository**

``` 
git clone git@github.com:amm834/aungmyatmoe.me.git
```

**Install Packages**

``` 
pnpm install
```

**Serve in Browser**

``` 
pnpm dev
```

**Entering Admin Mode**
Go to your served domain and then append `/admin`. It'll open up the admin panel.

## Production

You will need to attach `.env` variables correctly.

``` 
pnpm build
```
